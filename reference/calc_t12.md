# Calculate half-life based on two points

based on two sampling points (in same interval)

## Usage

``` r
calc_t12(t1, t2, y1, y2)
```

## Arguments

- t1:

  first sampling timepoint

- t2:

  second sampling timepoint

- y1:

  first sample value

- y2:

  second sample value

## Examples

``` r
calc_t12(3, 24, 30, 10)
#> [1] 13.24952
```
